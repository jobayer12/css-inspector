CSS Inspector Pro is committed to protecting your privacy. This extension operates entirely locally in your browser and does not collect, store, transmit, or track any data about you or your browsing.

  • ✗Does NOT collect any personal data

  • ✗Does NOT track your browsing history
  • ✗Does NOT store information about websites you visit

How the Extension Works

CSS Inspector Pro operates entirely locally in your browser:

Local Operation

All functionality runs directly in your browser on your device

No Server Communication

The extension does not communicate with any external servers

No Data Storage

We do not store any data about your usage or websites

Temporary Changes

CSS modifications exist only in your browser session

Permissions Explained

The extension requires the following Chrome permissions:

activeTab

  • Purpose: Access the current tab when you click the extension icon
  • Usage: Inject the CSS inspector interface into the active webpage
  • Data Access: Only accesses the current tab, only when activated

scripting

  • Purpose: Inject JavaScript and CSS into webpages
  • Usage: Add the inspector panel and modify CSS in real-time
  • Data Access: Only when you explicitly activate the extension

host_permissions

  • Purpose: Work on all websites
  • Usage: Enables you to inspect CSS on any webpage you visit
  • Limitation: Still requires you to click the extension icon to activate

CSS Inspector Pro explicitly does NOT:

  • ✗Collect personal information (name, email, etc.)
  • ✗Track your browsing activity
  • ✗Store cookies or use local storage for tracking
  • ✗Share data with third parties
  • ✗Display advertisements
  • ✗Use analytics or tracking tools
  • ✗Access passwords or sensitive information
  • ✗Modify webpages without your explicit action
  • ✗Run in the background without your knowledge
